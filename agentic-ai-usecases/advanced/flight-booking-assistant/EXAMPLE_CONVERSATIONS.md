## Key Conversation Patterns

### Success Pattern
1. Clear answers → Smooth progression
2. Expected format (cities, dates) → Fast processing
3. Confirmation → Immediate flight display
4. Single passenger → No age queries needed
5. One-way → Simplified journey details

### Error Recovery Pattern
1. Invalid input → Helpful suggestion with options
2. Repeat question → Clear rephrasing
3. Format error → Example shown
4. Clarification needed → Assistant guides next step
5. Eventually → Confirms and moves forward

### Passenger Types
- **Single Adult** - Quickest path (Example 1, 3)
- **Couple** - Parallel data entry (Example 2)
- **Family with Kids** - Additional age questions (Example 2)
- **Group/Corporate** - Multiple names (Example 2, 4)

## Tips for Best Experience

✅ Use clear city names (Mumbai, Delhi, Bangalore)  
✅ Use standard date formats (10th Feb, Feb 10, 2026-02-10)  
✅ Provide passenger count as "X adults, Y children"  
✅ Use valid email format (name@domain.com)  
✅ Use 10-digit phone numbers for India  
✅ Say "One-way" or "One way" for single-leg trips  
✅ Select flights by number (1, 2, 3) or "cheapest"
